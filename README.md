# WCWAVE
Scripts for iSNOBAL Model

## Naming convention

* initial:
   * z:
   * z_0: roughness_length
   * z_s: snow_depth
   * rho: snow_density
   * T_s_0: active_snow_layer_temperature
   * T_s: average_snow_covoer_temperature
   * h2o_sat: H2O_saturation
* precip:
   * m_pp: precipitation_mass
   * %_snow: percent_snow
   * rho_snow: precipitation_snow_density
   * T_pp: dew_point_temperature?
* input:
   * I_lw: thermal_radiation
   * T_a: air_temperature
   * e_a: vapor_pressure
   * u: wind_speed
   * T_g: soil_temperature
   * S_n: solar_radiation
